## ğŸ”§ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Neo-sk01/carbo-website.git
   cd carbo-website
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

4. Open your browser and navigate to:
   ```bash
   http://localhost:3000
   ```
